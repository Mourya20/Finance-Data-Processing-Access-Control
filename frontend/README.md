# Finance Data Frontend

This folder contains a React frontend app for the Finance Data Processing backend.

## Local setup

1. Navigate into the frontend folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Notes

- The frontend uses `VITE_API_URL` to connect to the backend.
- Backend endpoints are expected at `http://localhost:3000` by default.
- No backend files were modified; this app lives entirely in `frontend/`.
